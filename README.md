# Project Twelve Contracts Monorepo

## Usage

Requirement:

- Node >= 18
- pnpm >= 8

Clone the repository

```shell
$ git clone https://github.com/ProjectTwelve/contracts
```

Install dependencies

```shell
$ pnpm i
```

Run all test

```shell
$ pnpm test
```
